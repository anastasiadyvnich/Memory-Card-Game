The best way to run this project is by using the **Live Server** extension for Visual Studio Code. This simulates a real web server and automatically refreshes the browser whenever you save changes to the code.

## ðŸ›  Prerequisites

1.  **Visual Studio Code** (VS Code) installed.
2.  **Live Server** extension installed.

## ðŸ“¥ Step 1: Install the Extension

1.  Open **VS Code**.
2.  Go to the **Extensions** view (click the square icon on the left sidebar or press `Ctrl+Shift+X`).
3.  Type `Live Server` in the search bar.
4.  Find the extension created by **Ritwick Dey** and click **Install**.

## â–¶ï¸ Step 2: Launch the Game

1.  Open project folder in VS Code (`File` -> `Open Folder...`).
2.  In the file explorer (left sidebar), click on **`index.html`** to open it.
